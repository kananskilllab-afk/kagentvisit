# Plan 03-01 - Action Item Tracker Frontend

## Goal

Make action items usable from FormConfig-driven visit forms, visit review details, and dashboard open-item visibility.

## Tasks

- Add reusable `ActionItemTracker` supporting draft form mode and persisted visit mode.
- Add `action_items` to FormBuilder and DynamicField.
- Add Zod validation support for action item arrays.
- Normalize form-submitted action items in visit create/update handlers.
- Add `/api/visits/action-items/my-open` for dashboard widgets.
- Embed the tracker in `VisitDetailModal`.
- Add dashboard open action item cards for non-accounts roles.

## Verification

- `npm run build --prefix client`
- `npm test --prefix server`
